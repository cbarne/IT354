Values and Types

IsNaN


It is the asking of "Is Not A Number" which is automatically true on return unless stated false.

So for example one can not divide zero by zero since it becomes NaN.


Yet in the case of examples:


isNaN(5): Is 5 not a 5 so false
isNaN('a'):True since 'a' is not a number
isNaN(null)it will be false since it starts at 0.

Unary Operators


Plus - Minus


The plus helps to change the string of numbers, booleans, and null to numbers. It must be an arthmetic type, else it will result in an error.

The minus is the exact same except it is producing the negative of the operator.

So it will look something like this: "int x = -20; // x is -20 int y = +x; // y is -20"


Delete


It will return the boolean value if the property is properly deleted or it simply is returned. It wont work with variables since it returns boolean values, and it will not work with functions.

Hoisting


It is when the logical processing gets reversed. So what is at the bottom goes ot the top.
