Functions


Named Functions


Named functions are movable functions compared to its expressions.


Function Expressions


Expressions are unmovable functions that use a constant to define them.


Arrow Functions


Arrow functions are a short way to write a function and are still able to be movable depending on what its used for. It uses the 'fat arrow' as a definer instead of the function declaration.


Scope


Normal Scope

It is when one decides they are looking at the specific overview of a certain field of the project itself. So it could deal with names, a login feature, or a simple nav bar for examples of some different scope outlooks.


Nested Scope

Similar to a normal scope,but this is inside the normal scope. It targets a very specific function inside. Which can call to the outside but never inside.


Optional Arguments and Rest Parameters


Optional Arguments

An optional argument is one that can have inputted value in a random declared object. Then once it is called at the end; if it does not have the values specified then it will be dropped off of it.
*Order does matter when it is called.*


Rest Parameters

They allow the user to get a list and to basically reiterate it unlimitedly for arguments.




What is ECMAscript

Defined as a scripting-language specification by ECMA International.

Its first initial release was all the way back in 1997. It was designed by a man named Brendan Eich. It was created to help standardize the use of Javascript. Which has lead Javascript to become one of the most used implementers of the scripting language.


It is used for client side scripting and it is used in Node.js quite commonly. Since it is a scripting language, that means that it handles the fundemental actions that a user will input. Similar to a video game character needing to move.


Javascript uses the scripting language as a basis for how it runs its own guidelines. It is like saying you have a constitution to base your guidelines off of. Unless you're a pirate.

Since both Javascript aims to be very compatible, it also provides additional features not described in the guidelines. Those are posted in each new version update for more information.
