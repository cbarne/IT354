Modules


Importance of Them

Like everything you do in life; it can be defined into chapters. Well the coding world works the same way by dividing programs into modules. It helps with maintaining your program. So instead of changing a little thing all over. It allows you to do it all at once. It also allows for reusability. One can also "stitch" together modules known as module bundling. Which is the process of concatenating all of the files into a single one. Which some people also refer to it as minifying.

Web Font Issues

It all started out on a webpage that was telling users that Mitt Romney was not running for president. Problem became the word 'not' was unable to be seen. Quite an issue when viewers now think he is. It all began with the font being loaded with a seperate FOIT. Since browsers normally know to load multiple @font-face fonts, it was not thought to happen. These types of incidents can have dire effects on the world we live in today were information is spread like wildfire. I would not think to let the page load before I had already started scrolling to begin reading. So as a user that would be a crucial mishap. It would also ruin the credibility of the source that you read it from. These are ways to address the problem like the article states. I think the most simplist one is simple using one type of font for headers like that instead of relying on multiple types.
